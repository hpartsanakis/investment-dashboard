# Investment Dashboard

A full-featured portfolio tracker built with HTML, CSS, and JavaScript.

## Features

- Portfolio tracking
- Transaction system (BUY / SELL)
- Performance analysis (% and €)
- Asset allocation
- Watchlist
- LocalStorage persistence

## Tech Stack

- Vanilla JavaScript
- LocalStorage
- Responsive UI
